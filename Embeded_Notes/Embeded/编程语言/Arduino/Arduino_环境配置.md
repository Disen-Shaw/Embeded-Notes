# Arduino环境配置
## 下载arduino IDE
根据官网的[下载地址](https://www.arduino.cc/en/software)进行下载
下载后解压，完成安装
## 配置VScode开发环境
下载Arduino插件
![[Pasted image 20210326022503.png]]
安装后配置arduino的命令位置和工程位置
在建立一个新工程后，更改setting.json文件
添加或者更改："C\_Cpp.intelliSenseEngine": "Tag Parser"
建立工程完成，可以执行编译等操作，添加类不会报错，配置完成
