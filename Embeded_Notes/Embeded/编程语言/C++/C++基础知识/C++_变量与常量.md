# C++中的变量与常量

## C++中的变量

### C++中变量存在的意义

方便程序员管理和使用内存空间

### 变量的创建语法

```c++
变量类型 变量名 = 变量值;
变量类型 变量名;
```

> 变量名有相应的变量名规则，不可以随意取名

## C++中的常量

### C++中常量存在的意义

常量用于记录程序中不可以更改的数据

### 常量的定义方式

C++ 中有两种定义常量的方式：

- 通过 `define` 定义宏常量
  ```c++
  #define 常量名 常量值
  ```
- 通过 `const` 修饰定义变量
  ```c++
  /* 表示变量为常量，值不可以更改 */
  const 变量类型 变量名 = 变量值;
  ```
