# 通讯的基本原理
## 串行和并行通信
![[Pasted image 20210308222226.png]]

| 特性       | 串行通讯 | 并行通讯 |
| ---------- | -------- | -------- |
| 通讯距离   | 较远     | 较近     |
| 抗干扰能力 | 较强     | 较弱     |
| 传输速率   | 较慢     | 较高     |
| 成本       | 较低     | 较高     |

## 通信的速率

比特率：
每秒钟传输的二进制位数，单位为Bit/s
波特率：
每秒钟传输的码元的个数
