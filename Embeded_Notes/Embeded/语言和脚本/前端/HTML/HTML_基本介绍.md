# HTML 简介

## HTML 的含义

超文本标记语言(HyperText Markup Language 简称HTML)，用于 _创建网页_ 的标准 **标记语言**

## HTML 文件后缀名

- **.html**
- **.htm**

## 简单实例

```html
<html>

<head>
	<meta charset="UTF-8" />
	<meta name="viewport" content="width=device-width" />
	<title>邵笛笙的个人博客</title>
</head>

<body>
</body>
	<a href="https://www.baidu.com">百度网址<a>
</html>
```
