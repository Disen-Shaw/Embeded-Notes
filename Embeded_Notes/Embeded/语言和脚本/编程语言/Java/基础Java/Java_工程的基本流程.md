# Java 的基本流程

Java 程序的编写到执行有以下几个步骤

## Java 程序的编写

Hello World 示例：

```java
public class Main{
	public static void main(String[] args){
		System.out.println("Hello world");
	}
}
```

## 编译 Java 程序

使用 `javac` 命令编译程序

## 执行 Java 程序
使用 `java 类名` 执行程序

```log
 6107  vim Main.java
 6108  ls
 6109  javac Main.java
 6110  java Main
```
